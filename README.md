### 关键特性

* LTDC 提供了 24 位的并行数字 RGB（红、绿、蓝），传送的所有信号可直接与最高 1024x600 分辨率的 LCD 和 TFT 面板接口。
* VGA 输出, 支持刷新率不低于 20Hz: 640x480, 800x600
* 支持 RGB888 格式，向下兼容 RGB666、RGB565 等
